O problema

Você terminou a tarefa. Aí o review volta com 23 comentários.

O código funcionava — mas estava cheio de pontos cegos que você não viu: o caso de borda, a query insegura, o erro engolido em silêncio. A diferença entre júnior e pleno está em revisar o próprio código antes de expor. Este livro te ensina a treinar esse olhar — usando a IA como o seu primeiro revisor.

O método, na prática

A mesma IA. A diferença está em como você usa.

Não é pedir o código pronto. É escrever você mesmo e pedir pra IA tentar quebrar — antes do PR.

Você[cola, abre o PR sem entender]

Para quem este ebook é (e para quem não é).
